# Flutter Tutorial 🚀

Một repository học Flutter từ cơ bản đến nâng cao.

## Nội dung

- Cài đặt Flutter SDK
- Tạo ứng dụng Hello World
- Các widget cơ bản
- State management
- Navigation

## Yêu cầu

- Flutter SDK >= 3.0
- Dart SDK >= 3.0
- Android Studio / VS Code

## Bắt đầu

```bash
flutter pub get
flutter run
```
